Определить класс «Страна». 
Базовый класс определяет интерфейс для построения стран, но они могут быть специализированы для конкретного 
(Средиземье, Нарния, Вестерос, Страна Оз, Атлантида). 
В каждой стране: сколько государств (описание флагов (текст), столица, глава государства (или несколько)),
в каком фильме или книге упоминается, климат, наличие волшебства, какие расы населяют, какие мифические существа обитают. Использовать паттерн «Одиночка». 
Результат и периферийное оборудование записываются в отдельные выходные файлы.
